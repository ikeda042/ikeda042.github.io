About

研究現場から逆算して、必要なソフトウェアを形にする

広島大学での生物工学バックグラウンドと、University of Minnesota での CS 留学経験をベースに、 研究現場で本当に必要とされる機能を Web / API / ローカル環境へ落とし込む開発に取り組んでいます。

企画だけで終わらせず、導入障壁の低い UI、運用できる API、継続的に改善できる構成まで含めて設計することを重視しています。

Focus Research Software / Product Design / Backend Systems Approach 現場要件と運用要件を同時に満たす設計

Projects

LabSocket（研究室DXプロダクト）

これまでの経験を統合して、研究室 DX を行うプロダクトをローンチしました（広島大学スタートアップチャレンジ2025）。 研究現場のオペレーションを「ソフトウェア/API」に落とし込み、導入障壁を下げることを狙っています。

View details →

PhenoPixel（細胞画像解析ソフト）

顕微鏡データから細胞情報を高速に抽出し、夜間の自動解析まで含めたパイプラインを構築。 ローカルサーバーに展開し、ラボ内の全員が同一プログラム/APIへブラウザからアクセスできる形で運用しています。

Product page →
